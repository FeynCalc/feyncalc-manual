\section{\$LoadPhi}\label{loadphi}}

\texttt{\$LoadPhi} is a legacy switch for loading Phi. The modern way to
achieve the same effect is to evaluate
\texttt{\$LoadAddOns=\{\allowbreak{}"Phi"\}} before loading FeynCalc.
