\section{FactorFull}\label{factorfull}}

\texttt{FactorFull} is an option of \texttt{Factor2} (default
\texttt{False}). If set to \texttt{False}, products like
\texttt{(a-b) (a+b)} will be replaced by \texttt{a^2-b^2}.
