\section{PaVeLimitTo4}\label{pavelimitto4}\index{PaVeLimitTo4}}

\texttt{PaVeLimitTo4[\allowbreak{}expr]} simplifies products of
Passarino-Veltman functions and \(D\)-dependent prefactors by evaluating
the prefactors at \(D=4\) and adding an extra term from the product of
\((D-4)\) and the UV pole of the Passarino-Veltman function.

This is possible because the UV poles of arbitrary Passarino-Veltman
functions can be determined via \texttt{PaVeUVPart}. The result is valid
up to 0th order in Epsilon, i.e.~it is sufficient for 1-loop
calculations.

Warning! This simplification always ignores possible IR poles of
Passarino-Veltman functions. Therefore, it can be used only if all IR
poles are regulated without using dimensional regularization (e.g.~by
assigning gluons or photons a fake mass) or if it is known in advance
that the given expression is free of IR singularities.

The application of \texttt{PaVeLimitTo4} is equivalent to using the old
\texttt{OneLoop} routine with the flags \texttt{\$LimitTo4} and
\texttt{\$LimitTo4IRUnsafe} set to \texttt{True}.
