\section{Rename}\label{rename}\index{Rename}}

\texttt{Rename} is an option for \texttt{Contract}. If set to
\texttt{True}, dummy indices in \texttt{Eps} are renamed, using
\texttt{\$MU[\allowbreak{}i]}.
